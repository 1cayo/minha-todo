Sistema web de lista de tarefas. Dá pra adicionar e remover tarefas.

## O que usei

- React + Vite
- CSS puro

## O que aprendi

- JSX: escrever HTML dentro do JavaScript
- useState: guardar e atualizar dados na tela (a lista de tarefas e o input)
- map e filter: pra mostrar e remover itens da lista

## Como rodar

bash
npm install
npm run dev


Acessa http://localhost:5173 no navegador.
